## 📅 정기 모임 날짜

- 시간 : 매주 수요일 오후 7시
- 장소 : 학교 강의실 (예약: 윤태호)

## 🖋️ 진행 방식

- 주차 별 해당 인원이 6문제 선정 (현동 - 건화 - 우찬 - 세환 - 성현 - 태호 순)
- 명당 한 문제씩 전담해서 모임 시간에 설명
